# LocalScribe 0.2.2

## Highlights

- Auto-builds and starts the native ScreenCaptureKit helper from the backend when `Current Mac sound output` capture starts for the first time.
- Auto-starts Ollama from the backend when live recording begins with the Ollama cleanup backend selected.
- Tightens the live setup UI so Mac output capture and local AI preparation report backend-driven progress instead of asking users to open a separate terminal.
- Hides the Whisper model install progress card after a successful install instead of leaving stale download logs visible.
- Updates the native system-audio helper for the current AVFoundation API so the helper builds cleanly with the newer Apple Swift toolchain.

## Notes

- The first `Current Mac sound output` run can still take a moment while Swift builds the helper in the background.
- Automatic Ollama startup assumes `ollama` is installed and available on `PATH` or configured explicitly for the app environment.
