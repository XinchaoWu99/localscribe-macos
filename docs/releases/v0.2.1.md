# LocalScribe 0.2.1

## Highlights

- Reworked the live page flow so `Start Here` behaves like a compact onboarding step instead of a permanent large panel.
- Moved live session status into the transcript room for easier monitoring while transcribing and editing.
- Added a live input-level meter and quiet-input gain handling to improve transcription when browser audio arrives at a very low level.
- Added native macOS system-audio service wiring, helper-status reporting, and in-app setup guidance for current Mac sound output capture.
- Continued refinement of the live caption editor, speaker-aware turn handling, and local AI assistant cleanup flow.

## Notes

- Native `Current Mac sound output` still depends on building the ScreenCaptureKit helper once on the local machine.
- The helper build can remain blocked on machines where the active Swift compiler and active macOS SDK do not match.
