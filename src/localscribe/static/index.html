Quick Guide

Use LocalScribe like a live control desk

This tool is optimized for real-time transcription. Pick the audio source, choose a preset, create a session, then start the mic and watch the transcript update as the conversation unfolds.

1
Pick the audio path

Use live microphone for rooms and calls, Mac sound output for this Mac's playback, or browser audio when sharing a tab.

2
Choose a speaking scenario

Meeting, interview, podcast, and news presets adjust prompt bias and segment cadence automatically.

3
Start a live session

Create a session, start the mic, then review the speaker roster, markers, and transcript timeline below.
